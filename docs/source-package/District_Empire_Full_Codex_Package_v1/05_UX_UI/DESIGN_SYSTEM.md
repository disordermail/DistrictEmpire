# Design System

## Charakter

- czysty,
- neutralny,
- nowoczesny,
- lekki,
- finansowy,
- nieruchomościowy.

## Komponenty

- property card,
- map marker,
- metric chip,
- bottom sheet,
- timeline,
- price badge,
- owner badge,
- progress bar,
- auction card,
- alert card.

## Ikony

Każda kategoria ma ikonę.

## Typografia

Czytelna, bezszeryfowa, wysoki kontrast.

## Kolory

Kolory statusowe muszą działać także bez rozpoznawania barw.
