# UX Bible

## Kierunek

2D app-like.

Inspiracje:

- Airbnb,
- Otodom,
- aplikacje bankowe,
- mapy,
- dashboardy.

## Nawigacja

- Mapa
- Rynek
- Portfolio
- Zgłoszenia
- Firma

## Zasady

- bottom sheets,
- karty,
- zakładki,
- jedna ręka,
- mało modali,
- najważniejsze akcje w dolnych 60% ekranu.

## Główny ekran

Mapa z geolokalizacją i widoczną aktywnością graczy.
