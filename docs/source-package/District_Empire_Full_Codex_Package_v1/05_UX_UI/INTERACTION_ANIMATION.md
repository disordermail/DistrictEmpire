# Interaction Animation

## Zakup

- akt notarialny,
- wpisanie ceny,
- podpis,
- pieczęć,
- zmiana właściciela na mapie.

## Najem

- umowa,
- podpis najemcy,
- podpis gracza,
- klucz.

## Pełny budynek

- podświetlenie jednostek,
- ramka,
- nazwa firmy,
- odznaka.

## Otwarcie lokalu

- witryna,
- szyld,
- klienci,
- wzrost wpływu.

## Długości

- micro: 150–400 ms,
- zwykłe: 0.7–1.2 s,
- ważne: 1.5–3 s,
- prestiżowe: do 4 s.
