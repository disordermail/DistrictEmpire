# Economy Bible

## Cel

Dawać poczucie slot machine bez hazardu pieniężnego:

- częste małe sukcesy,
- rzadkie duże okazje,
- zmienność,
- pity timers,
- silna aspiracja.

## Krzywa

Wartość aktywów rośnie wykładniczo między 20 giełdami.
