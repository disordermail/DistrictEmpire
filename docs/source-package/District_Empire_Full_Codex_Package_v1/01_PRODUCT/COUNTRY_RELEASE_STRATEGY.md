# Country Release Strategy

## Model

Gra jest uruchamiana osobno dla każdego kraju.

Przykładowa kolejność:

1. Polska
2. Niemcy
3. Czechy
4. Austria
5. Hiszpania
6. Francja
7. Włochy
8. Wielka Brytania

## Country Pack zawiera

- kod kraju,
- języki,
- walutę,
- format liczb,
- miasta,
- dzielnice,
- mapy,
- typy budynków,
- lokalne biznesy,
- święta,
- sezonowość,
- podatki growe,
- ceny referencyjne,
- eventy kulturowe,
- lokalne ikony i landmarki.

## Izolacja rynku

Domyślnie gracz uczestniczy w rynku danego kraju.

Globalne rankingi mogą łączyć kraje, ale ceny i ekonomie pozostają lokalne.

## Tłumaczenia

Polski jest językiem źródłowym. Kolejne języki powstają jako kolumny w Excelu.
