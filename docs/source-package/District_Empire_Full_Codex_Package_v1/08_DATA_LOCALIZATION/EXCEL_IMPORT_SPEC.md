# Excel Import Specification

## Pliki

- Localization_Master.xlsx
- Country_Pack_Template.xlsx
- Content_Master.xlsx
- Economy_Config.xlsx
- LiveOps_Calendar_Template.xlsx

## Import

Excel -> walidacja -> JSON/CSV -> Unity assets lub backend.

## Walidacja

- unikalne ID,
- wymagane pola,
- poprawne enums,
- brak pustego polskiego tekstu,
- referencje do istniejących ikon,
- zakresy liczb,
- wersja danych.
