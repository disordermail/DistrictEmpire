# Audio and Haptics

- zakup,
- podpis,
- pieczęć,
- czynsz,
- podatek,
- aukcja,

## Haptics

- light: zwykłe potwierdzenie,
- medium: zakup,
- success: wygrana,
- warning: podatek,
- error: brak środków.

## Zasada
