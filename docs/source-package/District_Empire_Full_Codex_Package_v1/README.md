# District Empire — Full Codex Package v1

Kompletna dokumentacja docelowej wersji gry przed rozpoczęciem prac nad MVP.

## Założenia

- mobilna gra 2D na iOS i Android,
- orientacja pionowa, obsługa jedną ręką,
- główny ekran to mapa geolokalizacyjna,
- styl aplikacji Airbnb / Otodom / aplikacji bankowej,
- start w Polsce,
- późniejsze premiery osobno dla kolejnych krajów,
- silnik i systemy wspólne, content i konfiguracja per kraj,
- polski jest pierwszym językiem źródłowym,
- całość data-driven,
- ekonomia i czas autorytatywne po stronie serwera.
