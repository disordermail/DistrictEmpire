# Full Version to MVP Plan

## Zasada

Pełna dokumentacja definiuje system docelowy.

MVP implementuje wycinek bez zmiany architektury.

## MVP wycina

- 1 kraj: Polska,
- 1–2 miasta,
- 5 tierów zamiast 20,
- 5 typów nieruchomości,
- 20 biznesów,
- 20 eventów,
- 4 minigry,
- 1 ranking,
- rewarded ads,
- prosty sklep.

## MVP nie wycina

- lokalizacji,
- Country Pack,
- ledgeru,
- server time,
- data-driven content,
- pionowego UI,
- podstaw social visibility.
