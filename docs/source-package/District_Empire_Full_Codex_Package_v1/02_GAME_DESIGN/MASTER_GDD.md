# Master GDD

## Gatunek

Geolocation property tycoon / resource management / asynchronous social game.

## Core loop

1. Otwórz mapę.
2. Zobacz oferty i aktywność.
3. Kup nieruchomość.
4. Przygotuj ją.
5. Wystaw ogłoszenie.
6. Znajdź najemcę.
7. Pobieraj czynsz.
8. Płać podatki.
9. Rozwiązuj problemy.
10. Poprawiaj okolicę.
11. Awansuj na droższą giełdę.
12. Przejmuj większe aktywa.

## Filary

- mapa jako plansza,
- wzrost liczb,
- zmienny rynek,
- widoczność innych graczy,
- rozwój okolicy,
- krótkie decyzje,
- prestiż posiadania.

## Główne zasoby

- gotówka,
- kapitał netto,
- reputacja,
- wpływ,
- prestiż,
- czas,
- stan techniczny,
- zadowolenie najemców.
