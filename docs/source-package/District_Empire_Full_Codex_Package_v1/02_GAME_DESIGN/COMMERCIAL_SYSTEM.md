# Commercial System

## Zasada

Lokal użytkowy generuje większy przychód i znacznie większy podatek.

## Działalność

Gracz wybiera biznes z dużej listy.

## Nasycenie

Każda kategoria ma promień nasycenia.

Kolejny identyczny biznes daje mniejszy wpływ.

## Synergie

Niektóre biznesy wzajemnie się wzmacniają.

## Ryzyko

- słaby popyt,
- konkurencja,
- wyższy podatek,
- większy koszt otwarcia,
- możliwość bankructwa najemcy.
