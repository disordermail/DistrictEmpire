# Rental System

## Proces

1. Zakup.
2. Naprawy krytyczne.
3. Remont opcjonalny.
4. Cena najmu.
5. Publikacja.
6. Wyświetlenia.
7. Oględziny.
8. Zapytania.
9. Kandydaci.
10. Umowa.
11. Czynsz.

## Widoczne liczniki

- wyświetlenia,
- oględziny,
- zapytania,
- kandydaci.

## Czas

Od 1 godziny do 48 godzin czasu rzeczywistego.

## Popyt

Zależy od:

- ceny,
- jakości,
- okolicy,
- konkurencji,
- sezonu,
- reputacji,
- liczby mieszkańców,
- usług,
- stanu budynku,
- aktywności innych graczy.

## Najemca

Może:

- płacić terminowo,
- opóźnić czynsz,
- zgłosić awarię,
- odejść,
- polecić landlorda,
- negocjować.
