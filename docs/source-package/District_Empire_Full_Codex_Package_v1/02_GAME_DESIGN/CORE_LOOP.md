# Core Loop

## Codzienna sesja

System powinien przygotować 3–5 decyzji:

- nowa oferta,
- kandydat,
- awaria,
- podatek,
- aukcja,
- ranking,
- zdarzenie dzielnicowe.

## Sesja aspiracyjna

Gracz widzi:

- aktualny tier,
- kolejny tier,
- dwie droższe giełdy,
- brakujący kapitał,
- przykładowe aktywa do odblokowania.

## Nieobecność

Gra nie zatrzymuje się, ale:

- najem może trwać,
- podatki rosną,
- problemy mogą rozwiązać się automatycznie,
- gracz nie traci całego portfolio po krótkiej przerwie.
