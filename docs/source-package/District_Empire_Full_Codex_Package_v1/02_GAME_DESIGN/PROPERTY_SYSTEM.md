- kawalerka,
- mieszkanie,

- garaż,
- parking,
- działka,
- landmark.

## Jednostki i budynki

Budynek składa się z jednostek.

Gracz może:

- kupić pojedyncze mieszkanie,
- kupić lokal,
- kupić parking,
- kupić cały dom,
- zdobyć pełną kontrolę nad budynkiem.

## Full Building Ownership

Po spełnieniu reguły kontroli:

- publiczny znacznik,
- nazwa firmy,
- odznaka,
- animacja przejęcia,
- dostęp do inwestycji wspólnych,
- premia zarządcza.

## Starter Buildings

Każdy nowy gracz może kupić instancjonowane mieszkanie startowe.

Nie blokuje ono innych graczy i nie liczy się do przejęcia budynku.
